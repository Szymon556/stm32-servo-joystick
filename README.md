
# Sterowanie serwem z wykorzystaniem STM32

Projekt realizowany w ramach przedmiotu **Procesory Osadzone**.

## Opis
Sterowanie serwomechanizmem SG-90 za pomocą joysticka analogowego.
Odczyt realizowany przez ADC + DMA, sterowanie PWM (TIM1).

## Mikrokontroler
- STM32F411VET6

## Narzędzia
- STM32CubeMX
- STM32CubeIDE
- HAL

## Uruchomienie
Projekt można otworzyć bezpośrednio w STM32CubeIDE
lub wygenerować ponownie z pliku `.ioc`.
